# HR-Summary

This project is an end-to-end HR Analytics dashboard built using SQL and PowerBI. The goal of this project is to analyze employee and performance data to generate
meaningful make data-driven decisions.

The project covers:
~ Employee demographics and departmental distribution
~ Salary analysis and trends
~ Performance ratings and bonus distribution
~ Attendance tracking
~ Insights into wokrforce distribution by location and department

Technologies Used:
~ SQL: For data extraction, manipulation, aggregration, and joinning multiples tables
~ PowerBI: For interactive visualization, KPIs, and dashboard creation
~ CSV/Sample Datasets: Employees, Departments, Performance, Attendance

Learning Outcomes:
~ Hands-on experience with SQL queries, joins , aggregations, subqueries
~ Building interactive dashboards in PowerBI
~ Understanding HR Metrics and workforce analytics
~ Applying data cleaning and validation before visualization
~ Translating raw data into actionable insights
